Add per-page/post configurable WYSIWYG editors as a widget to your sidebar.

== Description ==
Different widget content on multiple pages? Exactly.

This plugin adds WYSIWYG editors on page/post level that can be filled with content.
Add the widget to your sidebar and start editing on page level!

== Installation ==
I recommend to install plugins via your WordPress Dashboard.

For manual installation:
Download mklasen's Dynamic Widget Plugin (http://wordpress.org/extend/plugins/mklasens-dynamic-widget/).

To install mklasen's Dynamic Widget follow these steps:

- Upload the folder \"mklasens-dynamic-widget\" to your WordPress plugin folder (/wp-content/plugins/)
- Activate the plugin through the 'Plugins' menu in WordPress
